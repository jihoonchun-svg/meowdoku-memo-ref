🐱 MeowDoku — 메모 모드 레퍼런스

🐱 0/6 🐟🐟🐟
색마다 고양이 한 마리 행·열마다 한 마리 8방향 인접 금지
📝 노트 모드 (실제 보드에 영향 없음)
조작
· 클릭 = X 토글 (여기 아님) / 고양이 칸 클릭 = 제거
· 더블클릭 = 고양이 배치
· (실기기) 스와이프 = 여러 칸 X 일괄 — 본 프로토타입은 클릭/더블클릭만
시각 위계 (겹쳐도 안 헷갈리게)
🐱 확정 고양이 — 불투명·선명
✕ 확정 X / ✕ 오답 잠금 X(빨강, 변경불가)
🐱 메모 고양이 — 반투명 + 점선
✕ 메모 X — 블루그레이 (가설 크로스아웃, 블루프린트 톤)
메모 모드 ON → 보드가 파란 테두리로 바뀜. 고양이를 놓으면 정답 여부와 무관하게 행·열·8방향에 메모 X가 무조건 퍼짐(가설). 하단 도구는 [모두지우기] 하나 — 메모 레이어만 싹 비우고 확정 보드는 보존.
